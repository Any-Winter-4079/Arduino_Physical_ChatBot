## Setup

1. First, clone (and cd into) the repository:

   ```
   git clone https://github.com/Any-Winter-4079/Arduino_Physical_ChatBot.git
   cd Arduino_Physical_ChatBot
   ```

2. Then, create a virtual environment:

   ```
   python -m venv venv
   ```

3. Activate it:

   - Windows: `venv\Scripts\activate`
   - macOS/Linux: `source venv/bin/activate`

4. And finally, install the dependencies in this new environment:

   ```
   pip install -r requirements.txt
   ```
